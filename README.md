# AlphaSolutions_ProjectCalculator_2025
Eksamensprojekt 2. semester KEA – Projektkalkulationsværktøj for Alpha Solutions

Udarbejdet af:
Bassim Benarab & Laman Huseynova

Formål:
Dette projekt er et webbaseret værktøj, der gør det muligt for Alpha Solutions at planlægge og beregne tidsforbrug på projekter.
Systemet skal kunne oprette projekter, delprojekter og opgaver, registrere timer og udregne det samlede tidsforbrug.

Teknologier:

Java Spring Boot

Thymeleaf

MySQL

GitHub Actions (CI/CD)

Azure App Service

Struktur:

/src → Kildekode

/resources → HTML + database scripts

/docs → Rapport og bilag
